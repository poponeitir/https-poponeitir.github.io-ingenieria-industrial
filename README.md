# Guía Completa de Ingeniería Industrial ⚙️

Este proyecto consiste en una **plataforma educativa interactiva** diseñada para centralizar conocimientos fundamentales, herramientas y perspectivas actuales sobre la carrera de Ingeniería Industrial. El sitio web utiliza un diseño moderno y minimalista con un enfoque en la experiencia del usuario (UX) y el rendimiento técnico.

## 🎯 Objetivo del Proyecto
El sitio busca servir como un recurso de referencia integral para estudiantes y profesionales, explicando cómo la ingeniería industrial optimiza sistemas integrados de personas, materiales, información y energía para maximizar la productividad y eficiencia[cite: 2].

## 🚀 Características Principales

*   **Fundamentos Transversales**: Definiciones conceptuales basadas en estándares de instituciones internacionales como el IIE[cite: 2].
*   **Mapa de Especialidades**: Desglose detallado de 9 áreas críticas, incluyendo Logística, Six Sigma, Manufactura, Ergonomía y Gestión Ambiental[cite: 2].
*   **Línea del Tiempo Interactiva**: Recorrido histórico desde la división del trabajo de Adam Smith (1776) hasta la integración de la IA en la Industria 4.0[cite: 2].
*   **Caja de Herramientas**: Explicación de metodologías clave como Lean Manufacturing (5S, Kanban, Kaizen), Simulación Industrial y Programación Lineal[cite: 2].
*   **Análisis Comparativo**: Tabla técnica que diferencia la Ingeniería Industrial de otras ramas (Mecánica, Sistemas, Civil) en términos de enfoque, gestión y finanzas[cite: 2].
*   **Recursos Dinámicos**: Buscador de glosario técnico, sección de preguntas frecuentes (FAQ) y enlaces a recursos externos recomendados[cite: 2].

## 🛠️ Stack Tecnológico

El proyecto está construido utilizando tecnologías web estándar para garantizar compatibilidad y velocidad:

*   **HTML5**: Estructura semántica avanzada para SEO y accesibilidad[cite: 2].
*   **CSS3 (Custom Properties)**: Uso de variables CSS (`:root`) para una gestión eficiente de la paleta de colores "Energetic Tech" (Negro, Ámbar, Rojo Coral)[cite: 2].
*   **JavaScript (Vanilla)**: Lógica nativa para la barra de progreso de lectura, animaciones de entrada (`Intersection Observer`), buscador de glosario y menús colapsables[cite: 2].
*   **Tipografía**: Integración de fuentes de alta legibilidad vía Google Fonts (`Bebas Neue`, `IBM Plex Sans`, `IBM Plex Mono`)[cite: 2].

## 📱 Diseño Responsivo
La arquitectura de la información se adapta automáticamente a diferentes dispositivos mediante un sistema de rejillas dinámicas (`grid-template-columns`) que reorganiza las secciones de 4, 3 o 2 columnas en una sola columna para móviles[cite: 2].
