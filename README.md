# Guía Completa de Ingeniería Industrial ⚙️

Este proyecto es una **plataforma educativa interactiva** diseñada para centralizar conocimientos fundamentales, herramientas y perspectivas actuales sobre la carrera de Ingeniería Industrial[cite: 2]. El sitio utiliza un diseño moderno con enfoque en la experiencia del usuario (UX) y una arquitectura de información sistémica[cite: 2].

## 🎯 Objetivo
Servir como recurso de referencia integral para estudiantes y profesionales, detallando cómo la ingeniería industrial optimiza sistemas complejos integrados por personas, materiales e información[cite: 2].

## 🚀 Características y Secciones
*   **Fundamentos Técnicos**: Definiciones basadas en estándares internacionales[cite: 2].
*   **Mapa de Especialidades**: Desglose de áreas como Logística, Calidad, Ergonomía y Manufactura[cite: 2].
*   **Caja de Herramientas**: Explicación de Lean Manufacturing (Kaizen, 5S), Six Sigma y Simulación[cite: 2].
*   **Análisis Comparativo**: Tabla técnica frente a otras ingenierías[cite: 2].
*   **Glosario Industrial**: Buscador dinámico de términos clave como OEE, JIT y SMED[cite: 2].

## ⚙️ Lógica y Funcionalidades Técnicas
El sitio incluye sistemas programados en **JavaScript nativo (Vanilla JS)** para mejorar la interactividad[cite: 2]:

*   **Buscador Reactivo del Glosario**: Función `filterGlosario(query)` que filtra conceptos en tiempo real comparando la entrada con atributos `data-term`[cite: 2].
*   **Navegación Dinámica**: 
    *   **Barra de Progreso**: Indicador `#progress-bar` que calcula el porcentaje de lectura según el desplazamiento vertical[cite: 2].
    *   **Scroll Spy**: El menú resalta la sección activa automáticamente monitoreando la posición del usuario[cite: 2].
*   **Animaciones Scroll Reveal**: Implementación de la API `IntersectionObserver` para cargar elementos visuales solo cuando entran en el viewport, optimizando el rendimiento[cite: 2].
*   **Menú Adaptable**: Lógica de gestión de estados para navegación fluida en dispositivos móviles[cite: 2].

## 🛠️ Stack Tecnológico
*   **HTML5**: Estructura semántica avanzada para SEO[cite: 2].
*   **CSS3**: Sistema de diseño basado en variables `:root` para una paleta "Tech-Dark"[cite: 2].
*   **JavaScript**: Lógica nativa de manipulación del DOM y APIs de visibilidad[cite: 2].
*   **Tipografía**: Integración de `Bebas Neue` e `IBM Plex` vía Google Fonts[cite: 2].
